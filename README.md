Automatically generates ssh key and adds to the authentication agent.
## Dependencies
* `xclip` for **Linux**
* `clip` for **Windows**
* `pbcopy` for **OS X**
* `bash` you can change **$profile** for other shells
## Installation
```sh
# Clone the repository
git clone https://github.com/mehmetumit/auto_ssh_gen.git
# Change directory
cd auto_ssh_gen
# Run the script
./auto_ssh_gen
```
